ANDROID MOVIE (TMDB) APP

Android take home test for Yassir

DESCRIPTION:
App fetches data from API and displays it in a card.

- Jetpack Compose
- Coroutines
- Retrofit
- LiveData
- MVVM architecture (using ViewModel)
